# 🚀 Prompt-Hub

A curated collection of high-quality AI prompts designed to improve productivity, learning, coding, content creation, research, and problem-solving across multiple Large Language Models (LLMs).

Whether you're using **ChatGPT**, **Claude**, **Gemini**, **Copilot**, or other AI assistants, Prompt-Hub provides ready-to-use prompts to help you get better and more consistent results.

---

## 📌 Features

- 📂 Organized prompt categories
- 🤖 Compatible with multiple AI models
- 📝 Easy-to-copy prompt templates
- 🚀 Beginner-friendly
- 💡 Regularly updated with new prompts
- 🎯 Prompts for real-world use cases

## 🚀 Getting Started

1. Browse the folders.
2. Choose a prompt.
3. Copy the prompt.
4. Paste it into your preferred AI assistant.
5. Modify it according to your requirements.

---

## 💡 Example Prompt

```text
Act as a senior software engineer.

Task:
Explain the following code step-by-step.

Requirements:
- Beginner-friendly explanation
- Mention time complexity
- Mention space complexity
- Suggest improvements
- Give optimized code if possible

Code:
<Paste your code here>
```
